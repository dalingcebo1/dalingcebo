# Font Files

## Required Fonts

This directory should contain the Neuropol font files from Typodermic Fonts.

### Neuropol (Regular weight)
Place the following files here:
- `Neuropol-Regular.woff2` (preferred)
- `Neuropol-Regular.woff` (fallback)

### Neuropol X (Wide variant for titles/headings)
Place the following files here:
- `NeuropolX-Regular.woff2` (preferred)
- `NeuropolX-Regular.woff` (fallback)

## License

Neuropol is a commercial font by Typodermic Fonts (https://typodermicfonts.com/).
Ensure you have the appropriate license before using these fonts in production.

## How to Obtain

1. Purchase or obtain a license for Neuropol from Typodermic Fonts
2. Download the font files
3. Convert to web formats (woff2, woff) if needed using tools like:
   - Font Squirrel Webfont Generator
   - CloudConvert
4. Place the converted files in this directory
5. The font-face declarations in globals.css will automatically reference them
